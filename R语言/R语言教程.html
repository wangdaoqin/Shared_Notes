R语言入门教程

1. R语言简介

R语言是一种用于统计分析和数据可视化的编程语言。它是开源的，拥有强大的包生态系统，广泛应用于数据科学、机器学习和统计分析领域。

2. 基础语法

2.1 变量赋值

# 基本赋值
x <- 5
y = 10

# 打印变量
print(x)
cat("y =", y, "\n")

2.2 数据类型

# 数值型
num <- 3.14

# 字符型
str <- "Hello, R!"

# 逻辑型
logic <- TRUE

# 查看数据类型
class(num)
class(str)
class(logic)

2.3 向量

# 创建向量
vec <- c(1, 2, 3, 4, 5)

# 向量运算
vec * 2
vec + 1

# 访问向量元素
vec[2]  # 访问第二个元素
vec[1:3]  # 访问前三个元素

3. 数据处理

3.1 数据框

# 创建数据框
df <- data.frame(
  name = c("张三", "李四", "王五"),
  age = c(20, 25, 30),
  score = c(85, 90, 78)
)

# 查看数据框
print(df)

# 访问列
df$name
df[, "age"]

3.2 数据导入

# 从CSV文件导入数据
data <- read.csv("data.csv")

# 从Excel文件导入数据
# 需要安装readxl包
# install.packages("readxl")
library(readxl)
data <- read_excel("data.xlsx")

4. 数据可视化

4.1 基本绘图

# 折线图
x <- 1:10
y <- x^2
plot(x, y, type = "l", main = "折线图", xlab = "X", ylab = "Y")

# 散点图
plot(x, y, type = "p", main = "散点图", col = "red")

# 直方图
data <- rnorm(1000)
hist(data, main = "直方图")

4.2 使用ggplot2

# 安装并加载ggplot2包
# install.packages("ggplot2")
library(ggplot2)

# 创建数据
df <- data.frame(
  x = 1:10,
  y = x^2
)

# 绘制散点图
ggplot(df, aes(x = x, y = y)) +
  geom_point(color = "blue", size = 3) +
  labs(title = "散点图", x = "X轴", y = "Y轴") +
  theme_minimal()

5. 函数定义

# 定义函数
my_function <- function(x, y) {
  result <- x + y
  return(result)
}

# 调用函数
my_function(3, 5)

6. 包的使用

# 安装包
install.packages("dplyr")

# 加载包
library(dplyr)

# 使用包中的函数
df %>% 
  filter(age > 20) %>% 
  select(name, score) %>% 
  arrange(desc(score))

7. 常用快捷键

  • Ctrl + Enter: 运行当前行
  • Ctrl + Shift + Enter: 运行整个脚本
  • Alt + -: 插入赋值运算符 (<-)
  • Ctrl + Shift + M: 插入管道运算符 (%>%)
