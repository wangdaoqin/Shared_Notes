如何更新网站

  1. 添加新笔记：在本地创建.md、.html或.pdf文件
  2. 分类管理：根据笔记内容，在index.html中对应分类下添加笔记卡片
  3. 更新链接：确保笔记卡片中的链接指向正确的文件路径
  4. 部署更新：
    • 使用Git提交更改：git add .
    • 提交信息：git commit -m "添加新笔记"
    • 推送到GitHub：git push
  5. 访问网站：GitHub Pages会自动更新，通常在几分钟内生效

支持的文件格式

  • Markdown (.md)：直接在浏览器中查看
  • HTML (.html)：直接在浏览器中查看
  • PDF (.pdf)：浏览器会自动打开或下载
